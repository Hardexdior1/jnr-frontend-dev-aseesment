# Nexora Invoice App

A React + Next.js invoice management app with line items, tax calculations, and invoice generation.

---

## Setup Instructions

1. **Clone the repository**  
```bash
git clone <your-repo-url>
cd frontend


2. **Install dependencies**  

npm install
# or
yarn

3. **Run the development server**  
npm run dev
# or
yarn dev


4. Open in browser
Visit http://localhost:3000


# Usage Notes

Enter data in the provided input fields.

Refresh the page before submitting to see localStorage persistence in action.

Add, edit, and delete invoice items using the respective buttons.

The VAT and WHT rates are editable.

Reduce the screen size or view on a mobile device to test responsiveness.

Click Print Invoice to print a copy of the invoice.

Click Generate Invoice after adding items to save and confirm the invoice.
